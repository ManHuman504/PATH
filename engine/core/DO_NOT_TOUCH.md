# ⚠️ CORE ENGINE - DO NOT TOUCH

## 🚫 ЗАПРЕЩЕНО РЕДАКТИРОВАТЬ БЕЗ РАЗРЕШЕНИЯ

Эта папка содержит **ядро системы PATH#** - микроядерную архитектуру движка.

### ❌ Что НЕЛЬЗЯ делать:

- ❌ Изменять файлы без явного разрешения владельца
- ❌ Добавлять новую логику в Engine
- ❌ Менять интерфейсы IModule, IExtension
- ❌ Модифицировать EventBus
- ❌ Трогать ModuleManager
- ❌ Менять систему команд (CommandBus)

### ✅ Что МОЖНО делать:

- ✅ Читать и изучать архитектуру
- ✅ Использовать API для разработки модулей
- ✅ Создавать новые модули в /packages/modules
- ✅ Создавать плагины в /packages/extensions
- ✅ Адаптировать модули под core (но не наоборот!)

### 🔧 Если нужно изменить core:

1. **Обсудить** с владельцем проекта
2. **Описать** причину изменения
3. **Получить разрешение**
4. **Документировать** изменения
5. **Протестировать** совместимость со всеми модулями

### 📋 Правило микроядерной архитектуры:

> **Core остается стабильным и минимальным.**  
> **Модули адаптируются под core, а не наоборот.**

### Если что-то не работает:

1. **Сначала** проверь модули и плагины
2. **Потом** проверь адаптацию к core API
3. **Только после** обращайся к владельцу за изменением core
