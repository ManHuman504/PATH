# Changelog

## [1.0.0] - 2026-02-08

### ✨ Добавлено
- Базовая архитектура модуля PathEngine 3D
- `PathViewport` - главный класс управления viewport
  - Интеграция Three.js (Scene, Camera, WebGLRenderer)
  - CSS2DRenderer для гибридного рендеринга
  - OrbitControls с инерцией и ограничениями
  - Система освещения (Ambient, Directional, Point lights)
  - Респонсивность и автоматическая адаптация размеров

- `PathNode` - класс интерактивной ноды
  - 3D-меш (BoxGeometry) с кастомизируемым материалом
  - HTML-контент через CSS2DObject
  - Методы трансформации: `setTransform()`, `setColor()`, `setHTML()`, `setVisible()`
  - Автоматическая генерация ID
  - Правильная очистка ресурсов

- `InteractionSystem` - полноценная система интерактивности
  - Raycasting для точной детекции объектов
  - Drag & Drop в 3D-пространстве
  - События: hover, click, drag start/middle/end
  - Визуальный feedback при наведении
  - Предотвращение конфликтов с контролами камеры

- `GridShader` - процедурный бесконечный фон
  - Vertex shader для позиционирования
  - Fragment shader с:
    - Точечной сеткой
    - Линиями сетки
    - Реакцией на позицию курсора (увеличение, яркость)
    - Ambient анимацией
  - Uniforms для динамического управления

- `initPathEngine()` - функция быстрой инициализации
  - Автоматическая настройка контейнера
  - Конфигурируемые параметры
  - Callback система для событий
