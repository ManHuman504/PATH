## 🐛 Известные ограничения

1. **Нет системы связей** - Пока только отдельные ноды
2. **Нет сериализации** - Сохранение/загрузка не реализовано
3. **Базовые стили** - Нужна более гибкая система тем
4. **Нет виртуализации** - Производительность падает при >1000 нод
5. **Простой raycasting** - Может быть неточным при наложении
