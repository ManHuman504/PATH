## 🎯 Ключевые компоненты

### PathViewport.ts (400+ строк)
**Главный класс системы** - управляет всем viewport

**Основные методы:**
- `constructor()` - инициализация Three.js
- `createNode()` - создание нод
- `removeNode()` - удаление нод
- `getNodes()` - получение всех нод
- `focusOnNode()` - фокус камеры
- `dispose()` - очистка ресурсов

**Приватные методы:**
- `createCamera()` - настройка камеры
- `createWebGLRenderer()` - WebGL рендерер
- `createCSS2DRenderer()` - CSS2D рендерер
- `createControls()` - OrbitControls
- `setupLighting()` - система освещения
- `createInfiniteGrid()` - процедурный фон
- `startAnimationLoop()` - главный цикл

### PathNode.ts (250+ строк)
**Класс интерактивной ноды** - гибридный рендеринг

**Публичные свойства:**
- `group` - THREE.Group контейнер
- `mesh` - 3D-меш корпуса
- `htmlObject` - CSS2DObject
- `htmlElement` - DOM элемент

**Методы:**
- `setTransform()` - позиция/вращение/масштаб
- `setHTML()` - HTML-контент
- `setColor()` - цвет корпуса
- `setVisible()` - видимость
- `getPosition()` - текущая позиция
- `dispose()` - очистка

### InteractionSystem.ts (350+ строк)
**Система интерактивности** - события мыши

**Основные методы:**
- `setupEventListeners()` - подключение обработчиков
- `updateMousePosition()` - нормализация координат
- `raycastNodes()` - поиск нод под курсором
- `onMouseMove()` - движение мыши
- `onMouseDown()` - нажатие кнопки
- `onMouseUp()` - отпускание кнопки
- `onClick()` - клик
- `setNodeHoverState()` - визуальный feedback
- `dispose()` - очистка

### gridShader.ts (100+ строк)
**Процедурные шейдеры** - бесконечный фон

**Компоненты:**
- `gridVertexShader` - позиционирование
- `gridFragmentShader` - рендеринг сетки
  - `gridPattern()` - точечная сетка
  - `gridLines()` - линии сетки
  - Эффект курсора
  - Ambient анимация
